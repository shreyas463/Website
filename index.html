BioSearch-CPP

This project develops a specialized search engine for the Cal Poly Pomona Biology Department, enabling users to find faculty members based on research interests, expertise, and other attributes. It crawls faculty web pages, extracts structured profile data, and processes text using lemmatization. The search functionality leverages TF-IDF with cosine similarity ranking, includes spell-checking, and presents paginated results with clickable URLs. Data is persistently stored in MongoDB for efficient retrieval.

Python MongoDB TF-IDF Web Crawling Lemmatization Natural Language Processing
